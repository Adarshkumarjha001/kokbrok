📝 Level 0 DFD Description

The Context Diagram (Level 0 DFD) shows the Kokborok NLP System as a single process interacting with external entities. This high-level view illustrates:

  • User: Kokborok speakers who input text for analysis
  • Kokborok NLP System: The central processing system that performs POS tagging
  • Hugging Face Model Hub: External repository providing the trained AI model
  • Data Flows:
    • Input: Kokborok text from user
    • Output: Tagged results with POS labels
    • Model Request: System fetches model weights
    • Model Response: Hub provides model configuration
